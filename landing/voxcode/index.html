FAQ

Questions?

Does it work offline?
Yes. Whisper runs 100% locally. Pair it with Ollama and VoxCode works with zero internet connection. For Claude/GPT-4, you need an API key and internet.
What languages does it support?
Speech recognition works in 99 languages. Code generation works in any language your AI model supports — Python, JS, TypeScript, Go, Rust, Java, C++, SQL, Bash, and more.
How accurate is the transcription?
With Whisper large-v3, accuracy is 95-99% for English. You can say code naturally — "for loop that iterates over users" not "f-o-r space u-s-e-r space i-n space u-s-e-r-s".
Is my code sent anywhere?
Only to the AI provider you choose. With Ollama, nothing leaves your machine. With Claude or GPT-4, your prompts go to Anthropic/OpenAI — same as using their APIs directly.
What's the license?
Pro is a lifetime personal license. Use it on all your machines. The free version is open source on GitHub.
I bought it. How do I get my license?
Your license key is emailed to you immediately after purchase. Paste it into VoxCode with voxcode activate YOUR-KEY.
Does it work on Windows / Mac / Linux?
Yes — all three. Requires Python 3.10+ and a microphone.
