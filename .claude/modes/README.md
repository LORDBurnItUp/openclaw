# Claude Code Sub-Agent Modes

This directory contains 100 specialized sub-agent configurations for different development domains.

## Usage

Each sub-agent mode can be activated to provide specialized assistance for that domain.

## Categories

- [Frontend](./frontend/) - 20 agents for frontend development
- [Backend](./backend/) - 20 agents for backend development  
- [DevOps](./devops/) - 15 agents for DevOps tasks
- [Database](./database/) - 10 agents for database work
- [Security](./security/) - 10 agents for security
- [AI/ML](./ai-ml/) - 10 agents for AI/ML development
- [Mobile](./mobile/) - 10 agents for mobile development
- [General](./general/) - 5 general-purpose agents

Total: 100 specialized agents
